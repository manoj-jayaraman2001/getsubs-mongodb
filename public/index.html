Subscriber API Schema Docs

Base URL

https://getsubs-mongodb.onrender.com/subscribers

Get all subscribers

Request

  • Method: GET

Response

response with an array of subscribers (an object)

    Status Code: 200 OK
    Content-Type: application/json
  
    Body:
    [
      {
        "id": "1",
        "name": "John Doe",
        "subscribedChannel": "Channel A"
      },
      {
        "id": "2",
        "name": "Jane Smith",
        "subscribedChannel": "Channel B"
      },
      // ...
    ]

Error Responses

  • 500 Internal Server Error

Get subscriber names

Request

  • Method: GET
  • URL:

    https://getsubs-mongodb.onrender.com/subscribers/names

Response

response with an array of subscribers (an object) with only two fields "name" and "subscribedChannel" 

    Status Code: 200 OK
    Content-Type: application/json
  
    Body:
    [
    {
        "name": "Jeread Krus",
        "subscribedChannel": "CNET"
      },
      {
        "name": "John Doe",
        "subscribedChannel": "freeCodeCamp.org"
      }
      // ...
    ]

Error Responses

  • 500 Internal Server Error

Get subscriber by ID

Request

  • Method: GET
  • URL:

    https://getsubs-mongodb.onrender.com/:id
    (Replace :id with the actual ID of the subscriber)

    Example

    https://getsubs-mongodb.onrender.com/subscribers/64af942af0d7810033793b38

Response

response with an array of subscribers (an object) with only two fields "name" and "subscribedChannel" 

    Status Code: 200 OK
    Content-Type: application/json
  
    Body:
    {
      "id": "64af942af0d7810033793b38",
      "name": "Jeread Krus",
      "subscribedChannel": "CNET"
    }

Error Responses

  • 400 Not Found: This response is returned when no data is found with the given ID.
  • 500 Internal Server Error

If a request or Route is incorrect, response is :

404 Not Found
